>> 新建
    Shift + A: 新建物体
    Shift + D: 移动并复制
    Alt + D: 关联复制(物体共享数据)

>> TRS
    Shift + S: 打开菜单
    Shift + 空格: 唤出菜单
        G: 移动
        R: 旋转
        S: 缩放
        T: TRS

>> 平移
    G: 进入移动模式
        X: 沿X轴移动
        Y: 沿Y轴移动
        Z: 沿Z轴移动
        鼠标中键: XYZ吸附
    Alt + G: 原点移到游标
    选择磁铁, 并选择吸附方式
        独立吸附 - 到面投射
        独立吸附 - 到面最近

>> 旋转
    R: 进入旋转模式
    Alt + R: 清除旋转

>> 缩放
    S: 进入缩放模式
    Alt + S: 清除缩放
    ！！！！！！！！！！一定要应用缩放

>> 删除
    X: 删除物体

>> 连接
    Ctrl + J: 连接两个物体

>> 父级化
    Ctrl + P: 父级化
        先选父级
        后选子级
        后选子级
        后选子级
    基于物体: 父动, 子不动
    基于顶点：父动, 子动 + 不转
    基于顶点(三点): 父动, 子动 + 旋转


>> 关联材持
    Ctrl + L
